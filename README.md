# DDAM Library System

Company internal knowledge management system for R&D materials, project information, service/model documentation, and shared resources.

## Purpose

This system is intended to collect company internal materials, research documents, project information, and technical resources in one centralized place.

## Main Features

- R&D material library
- Project information repository
- Service/model catalog
- Document upload and search
- People and owner information
- Tags and categories
- Access control
